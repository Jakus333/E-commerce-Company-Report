# E-commerce Company Report
A report created to help a business identify it's strongest market, product lines, and future direction. Data was cleansed and tranformed in MySQL and the report was created in Power BI. The report pages are: High-level overview, Product-focus, Country-focus, and Scenario generator (allowing the user to see a forecast of future performance based on selected items. 

Time frame: 1 month (October - November 2025)

Software:

Power BI 
MySQL 
Microsoft Excel

Data Source: 
https://www.kaggle.com/datasets/gabrielramos87/an-online-shop-business
